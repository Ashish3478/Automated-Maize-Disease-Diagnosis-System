# Automated Maize Disease Diagnosis System

## 📌 Overview
Developed an automated deep learning pipeline to accurately identify and classify maize leaf diseases from digital images. This system is designed to replace slow and error-prone manual field inspections with rapid, AI-driven diagnostics.

## 🚀 Key Features & Methodology
* **Multi-Class Identification:** Built and trained Convolutional Neural Networks (CNNs) to classify plant leaves into four distinct categories: 
  * Healthy
  * Common Rust
  * Northern Leaf Blight
  * Gray Leaf Spot
* **Rigorous Benchmarking:** Evaluated multiple deep learning architectures (including VGG16, MobileNetV2, and ResNet101) against traditional machine learning models to optimize predictive performance.

## 📊 Performance
* **Peak Accuracy:** Achieved a **97.6%** classification accuracy using **ResNet101**.
* **Technical Optimization:** Successfully mitigated vanishing gradients and enabled the model to learn highly complex disease patterns across diverse visual datasets.

## 🌍 Impact & Deployment
Architected a highly scalable solution explicitly designed for mobile smart farming applications. This enables farmers to:
* Detect diseases at an early stage
* Minimize widespread crop loss
* Optimize and reduce unnecessary pesticide usage
