# Automated-Maize-Disease-Diagnosis-System
Developed an automated deep learning pipeline to accurately identify and classify maize leaf diseases from digital images, replacing slow and error-prone manual field inspections.
Built and trained Convolutional Neural Networks (CNNs) to classify plant leaves into four distinct categories: Healthy, Common Rust, Northern Leaf Blight, and Gray Leaf Spot.
Benchmarked multiple deep learning architectures (including VGG16, MobileNetV2, and ResNet101) against traditional machine learning models to optimize predictive performance.
Achieved a peak classification accuracy of 97.6% using ResNet101, successfully mitigating vanishing gradients and learning complex disease patterns.
Architected a highly scalable solution designed for mobile smart farming applications, enabling farmers to detect diseases early, minimize crop loss, and optimize pesticide usage.
